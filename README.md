# 1. 계획
1. 어떤 언어를 사용할 것인가?
  - JS , HTML , CSS , React

2. 무엇을 만들 것인가?
  - 내가 평소에도 계속 쓸 수 있고, 지인들도 사용할 수 있는 운동 관련 사이트

3. 만들기 전 계획수립
  - 기존에 있는 사이트의 문제점을 보완.
  - 기존 사이트에 존재하지 않아서 필요로 했던 사항 추가.
  - 피드백을 수용하여 +할 부분 , -할 부분 참고.
  - 모바일에서도 쾌적한 사용환경.
